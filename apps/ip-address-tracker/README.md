## Live demo

https://fm-ip-address-tracker.netlify.app/

## How to run

To run it locally, create a `.env.dev` file:

```
IPIFY_TOKEN=your-ipify-token
MAPBOX_TOKEN=your-mapbox-token
```

Get your tokens:

- https://geo.ipify.org/
- https://www.mapbox.com/maps/
